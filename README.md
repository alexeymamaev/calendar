# Календарь беременности

Самообновляющаяся одностраничная вёрстка: акушерский срок по неделям, зоны
доношенности (ВОЗ/ACOG), важные даты. Весь расчёт — на клиенте от опорной даты,
поэтому «сегодня» и текущая неделя пересчитываются при каждом открытии, без сборки.

- **Демо:** https://alexeymamaev.github.io/calendar/
- **Настройка:** один блок `CFG` в начале `<script>` в `index.html` (дата старта отсчёта,
  дата зачатия). `showName`/`name` оставлены пустыми намеренно — версия обезличена.

Статика, без зависимостей. GitHub Pages из ветки `main`, корень.
